Recipe Advisor using K-Means

I developed a recipe recommendation app that leverages Natural Language Processing (NLP) using SpaCy and K-Means clustering to provide ingredient-based recipe suggestions. The app analyzes ingredient lists and uses K-Means clustering to group similar recipes, ensuring personalized and relevant recommendations. The user interface is built with Streamlit, providing an interactive and intuitive experience where users can input available ingredients and receive real-time suggestions. With a high 96.42% precision and recall score, the app delivers accurate and reliable recipe suggestions. The project integrates text processing, clustering algorithms, and a responsive UI, making it a valuable addition to any portfolio, demonstrating my proficiency in NLP, machine learning, and web development.

Printer Service Application

I developed a full-stack Printer Service Application to streamline printer service scheduling, utilizing React.js for the front-end, Spring Boot for the back-end, Node.js for API handling, and MySQL for database management. Traffic Sign Classification

I developed a Traffic Sign Classification system by synthesizing and training a Convolutional Neural Network (CNN) model using TensorFlow to classify 42 distinct traffic signs. The model was trained on a large dataset of traffic sign images, achieving high accuracy in recognizing various road signs. To enhance user interaction, I integrated the trained model into a real-time graphical user interface (GUI) built with Tkinter, allowing users to upload images of traffic signs and receive instant classifications. This project demonstrates my expertise in machine learning, deep learning, and GUI development, showcasing the ability to combine advanced neural network models with interactive, user-friendly applications.

GitHub Repository

Face Mask Detection

I developed a Face Mask Detection system using MobileNetV2 and Convolutional Neural Networks (CNN), integrated with OpenCV for real-time face detection and mask compliance monitoring. Leveraging Keras for model training, I fine-tuned MobileNetV2 to classify images of individuals wearing face masks or not, achieving an impressive 98% detection accuracy. The system utilizes live video feeds captured through a webcam, processed by OpenCV to detect faces in real-time and check for mask compliance. A user-friendly interface was built using Tkinter, allowing users to easily interact with the system. This project highlights my proficiency in deep learning, computer vision, and real-time application development.
